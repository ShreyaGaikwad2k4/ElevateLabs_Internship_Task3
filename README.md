# Simple REST API with Express.js

This project is a beginner-friendly REST API built using Node.js and Express.js. It demonstrates REST API fundamentals, including HTTP methods, JSON handling, middleware, and basic CRUD operations.


## Features
- Express.js setup
- REST API architecture
- CRUD operations (Create, Read, Update, Delete)
- JSON request/response handling
- Middleware usage
- Simple, clean project structure


## Project Structure
project-folder/
 ├── server.js
 ├── package.json
 └── README.md


## How to Run the Project
1. Install dependencies:
   npm install

2. Start the server:
   npm start

3. The server will run at:
   http://localhost:3000


## Endpoints Overview
GET /users – Returns all users  
GET /users/:id – Returns a user by ID  
POST /users – Adds a new user  
PUT /users/:id – Updates a user  
DELETE /users/:id – Deletes a user
